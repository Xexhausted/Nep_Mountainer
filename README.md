# NepMountainer Resort - Himalayan Luxury

A premium, responsive static website for **NepMountainer Resort**, a luxury destination located in the heart of the Himalayas (Pokhara, Nepal). This site is designed to provide a seamless user experience for guests looking to explore rooms, book stays, and discover mountain experiences.

## 🏔️ Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop devices.
- **Room Showcase**: Detailed listings of Single, Double, and Deluxe bedrooms with luxury amenities.
- **Himalayan Experiences**: Curated sections for trekking, local culture, and wellness.
- **Interactive Booking**: A robust reservation form with real-time validation.
- **Contact & Support**: Integrated contact form for guest inquiries.
- **Google Forms Integration**: Both Booking and Contact forms are pre-configured to send data directly to Google Sheets for easy management.
- **Modern UI/UX**: Smooth scroll animations, sticky navigation, and a clean "mountain-luxury" aesthetic.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3 (Tailwind-inspired)**: Custom utility-first styling for high performance.
- **JavaScript (Vanilla)**: Lightweight interactivity, form validation, and scroll effects.
- **Google Forms**: Backend-less data collection for static hosting.
- **Lucide Icons**: Clean, modern iconography.

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/nepmountainer-resort.git
   ```
2. **Open the site**:
   Simply open `index.html` in any modern web browser.

## 🌐 Hosting on GitHub Pages

This site is perfectly suited for **GitHub Pages**:
1. Push this code to a public GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/ (root)` folder.
4. Click **Save**. Your site will be live at `https://your-username.github.io/nepmountainer-resort/`.
